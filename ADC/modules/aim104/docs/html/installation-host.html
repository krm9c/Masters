A.2. Installation - host

  1. Copy the include/linux/arcom/aim104.h header file to the /usr/include/linux/arcom directory.

    Alternatively, the header file can be installed as prefix/linux/arcom/aim104.h provided the appropriate command line parameter (-Iprefix) is passed to the compiler.

  2. (Optional) If you require the C library:

    1. Copy the include/arcom/libaim104.h header file to the /usr/include/arcom directory.

      Alternatively, the header file can be installed as prefix/arcom/libaim104.h provided the appropriate command line parameter (-Iprefix) is passed to the compiler.)

  3. (Optional) If you require the Java or GCJ library:

    1. You'll need to make the java/aim104.jar JAR file available to your java compiler (by setting the classpath) so it can look up the definitions of the classes. How exactly this is done depends on the particular compiler/environment
