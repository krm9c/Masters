In addition to the low level system call interface to the kernel drivers there is a C library (libaim104) providing a number of functions to access the different boards.

The function declarations are in arcom/libaim104.h. The library itself is the shared library libaim104.so.x.y.z.

Before any of the function can be used the device must by opened with the open() system call. The file descriptor returned by this call is then passed to each of the functions.

The device must be closed after use with the close() system call.
