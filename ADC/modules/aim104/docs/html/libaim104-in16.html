AIM104-IN16

Name

aim104_in16_inputs -- access an AIM104-IN16 board

Synopsis

#include <arcom/libaim104.h>

int aim104_in16_inputs(int fd, int channel);

Description

aim104_in16_inputs() returns the state of 8 of the digital I/O lines. A channel of 0 or 1 corresponds to lines 0-7 or 8-15 respectively.

Return Values

All functions return 0 or an 8 bit unsigned value on success. A negative value is returned when an error occurs.

The error codes are:

enum {
    AIM104_SUCCESS = 0,        /* Success -- no error occurred. */
    AIM104_EBAD_CHANNEL = -10, /* channel was not 0 or 1. */
    AIM104_EIO = -20,          /* Low level I/O error -- check errno for details. */
};
