1.2. AIM104 board modules

All the boards are accessed via character devices with major number 240. Each board module can handle up to 8 boards of the same type with a unique minor number for each board.

Each board module (such as aim104-relay8.o and aim104-io16.o) accept the same module parameters and have largely similar character device interfaces.

1.2.1. Module parameters

Table 1-2. Module parameters

ParameterAccepted valueDescription
io_baseComma separated list of integers.Base I/O address of each board.

Example 1-2. Loading the aim104-relay8.o module

If we have two AIM104-RELAY8/IN8 boards with base addresses jumpered at 180H and 184H.

insmod aim104-relay8 io_base=0x180,0x184

On some non-x86 platforms it is necessary to offset the address by some amount in order to access the ISA I/O address space. Please consult the relevant Quick Start manual.
