AIM104-OUT16

Name

aim104_out16_enable_outputs, aim104_out16_set_all, aim104_out16_set_masked, aim104_out16_output_status -- access an AIM104-OUT16 board

Synopsis

#include <arcom/libaim104.h>

int aim104_out16_enable_outputs(int fd, int enable);

int aim104_out16_set_all(int fd, int channel, unsigned char set);

int aim104_out16_set_masked(int fd, int channel, unsigned char mask, unsigned char set);

int aim104_out16_output_status(int fd, int channel);

Description

aim104_out16_enable_outputs() enables (enable is true/non-zero) or disables (enable is false/zero) all the outputs.

The following functions operate on 8 lines at a line. A channel of 0 or 1 corresponds to lines 0-7 or 8-15 respectively.

aim104_out16_set_all() sets 8 outputs to the state given by set.

aim104_out16_set_masked() sets 8 outputs to the state given by set except that outputs not included in the mask bit-mask are not changed.

aim104_out16_output_status() returns the current state of 8 outputs.

Return Values

All functions return 0 or an 8 bit unsigned value on success. A negative value is returned when an error occurs.

The error codes are:

enum {
    AIM104_SUCCESS = 0,        /* Success -- no error occurred. */
    AIM104_EBAD_CHANNEL = -10, /* channel was not 0 or 1. */
    AIM104_EIO = -20,          /* Low level I/O error -- check errno for details. */
};
