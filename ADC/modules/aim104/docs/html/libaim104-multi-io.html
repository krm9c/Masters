AIM104-MULTI-IO

Name

aim104_multi_io_inputs, aim104_multi_io_ADC, aim104_multi_io_DAC -- access an AIM104-MULTI-IO board

Synopsis

#include <arcom/libaim104.h>

int aim104_multi_io_inputs(int fd);

int aim104_multi_io_ADC(int fd, int channel, int single_ended);

int aim104_multi_io_DAC(int fd, int channel, unsigned short output);

Description

aim104_multi_io_inputs() returns the status of the 8 digital input lines.

aim104_multi_io_ADC() return a value read from one of the ADC inputs. For an ADC input connected in single ended mode single_ended must be set to true/non-zero and channel must be between 0 and 15 inclusive. For an ADC input connected in differential mode single_ended must be set to false/zero and channel must be between 0 and 7 inclusive.

aim104_multi_io_DAC() writes the 12 bit unsigned value (0-4095) output to DAC channel 0 or 1.

Return Values

All functions return 0 or an 8 bit unsigned value on success. A negative value is returned when an error occurs.

The error codes are:

enum {
    AIM104_SUCCESS = 0,        /* Success -- no error occurred. */
    AIM104_EBAD_CHANNEL = -10, /* channel was not 0, 1, 2, or 3. */
    AIM104_EIO = -20,          /* Low level I/O error -- check errno for details. */
    AIM104_ERANGE = -30,       /* Value to write to DAC was greater than 4095. */
};
