AIM104-RELAY8/IN8

Name

aim104_relay8_enable_relays, aim104_relay8_set_all, aim104_relay8_inputs, aim104_relay8_relay_status -- access an AIM104-RELAY8/IN8 board

Synopsis

#include <arcom/libaim104.h>

int aim104_relay8_enable_relays(int fd, int enable);

int aim104_relay8_set_all(int fd, unsigned char set);

int aim104_relay8_set_masked(int fd, unsigned char mask, unsigned char set);

int aim104_relay8_inputs(int fd);

int aim104_relay8_relay_status(int fd);

Description

aim104_relay8_enable_relays() enables (enable is true/non-zero) or disables (enable is false/zero) all the relays.

aim104_relay8_set_all() sets all 8 relays to the state given by set.

aim104_relay8_set_masked() sets the all 8 relays to the state given by set except that relays not included in the mask bit-mask are not changed.

aim104_relay8_inputs() returns the state of the 8 digital I/O lines.

aim104_relay8_relay_status() returns the current state of the 8 relays.

Return Values

All functions return 0 or an 8 bit unsigned value on success. A negative value is returned when an error occurs.

The error codes are:

enum {
    AIM104_SUCCESS = 0, /* Success -- no error occurred. */
    AIM104_EIO = -20,   /* Low level I/O error -- check errno for details. */
};
